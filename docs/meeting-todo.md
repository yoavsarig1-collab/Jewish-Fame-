## ⚠️ הדבר הקריטי ביותר: פער בין שמות הקטגוריות לתקנון

בדקתי מול התקנון ב-NotebookLM. **השמות החדשים שביקשת לא תואמים לתקנון הרשמי.**

| מה ביקשת (יושם בענף) | מה כתוב בתקנון |
|----------------------|----------------|
| מלחין / משורר ישראלי (06) | אין. יש "מלחין/מנצח: קלאסי, מחזמר, אופרה, פופ/רוק (ברוטציה)" — קטגוריה אחת |
| מלחין / משורר עולמי (07) | אין. בתקנון 07 = "להקה (מצמד ומעלה)" |
| מחזמר (08) | לא עצמאי — משולב בקטגוריית המלחין הרוטציונית |
| פסנתרן (11) | אין. בתקנון 09–10 = "התחום הקלאסי: שני נגנים" |
| מפיק (12) | משולב ב-"התחום הפופולרי: נגן/מפיק (שני פרסים)" |
| חזן (03) | בתקנון: "חזן/פייטן (אשכנזי או מזרחי)" |

**בנוסף — בתקנון יש קטגוריה שלא קיימת באתר:** "זמר/ת ים תיכוני (מישראל ומארצות המזרח)".
**ועדיין פתוח:** קטגוריית "משורר/פזמונאי".

➡️ **החלטה לסגור עם ענת/הוועדה:** האם משנים את התקנון (השמות החדשים הם הכיוון), או שהאתר צריך לחזור לשמות התקנון? הקטגוריות הן הבסיס לכל השאר.
